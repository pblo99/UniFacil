# UniFácil

Protótipo acadêmico de uma plataforma web responsiva para centralizar informações universitárias, como disciplinas, materiais, grupos, eventos, avaliações, fórum, chat e avisos em um único ambiente digital.

## Objetivo

Demonstrar a proposta de um hub acadêmico em formato de aplicativo web, com foco em organização, usabilidade e acesso rápido às informações do aluno.

## Funcionalidades

- Login simulado
- Painel inicial do aluno
- Disciplinas
- Avaliação de disciplinas
- Grupos acadêmicos
- Fórum / Q&A
- Chat simulado
- Materiais
- Eventos
- Perfil do aluno
- Persistência local com localStorage

## Como executar

```bash
npm install
npm run dev
```

## Como gerar versão final

```bash
npm run build
```

## Observação

Este projeto é um protótipo acadêmico e utiliza dados simulados.
